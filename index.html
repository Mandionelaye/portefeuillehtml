Le développeur informatique est chargé de construire des programmes informatiques pour ses clients (faire du développement web). Un développeur informatique spécialisait dans la création d'application pour les smartphones et tablette Le webmaster est le gestionnaire d'un site internet : il est en charge de son bon fonctionnement au quotidien il est chargé de réaliser les éléments graphiques d'un site web comme les illustrations, les bannières, les animations flash, les fonds d'écran et parfois même les logos. L,expert en référencement est chargé d'accroître la visibilité et la présence sur le web des sites internet.
